## 1. 项目概述

本项目是一个基于 STM32H723 (DM-MC02 开发板) 的**双轮腿式平衡机器人**控制系统。机器人采用五连杆腿部结构 + 轮毂电机的构型，每条腿由 2 个 DM4310 关节电机和 1 个 DM6215 轮毂电机驱动，共 6 个电机。

**核心控制思路**：将五连杆等效为虚拟单腿（VMC），建立倒立摆动力学模型，使用 **LQR** 实现机体俯仰平衡和速度/位移控制，使用多组 **PID** 实现腿长控制、横滚角补偿、偏航角转向和防劈叉补偿，使用**卡尔曼滤波器**融合加速度计和编码器数据估计车体速度和位移。

## 4. 控制算法详解

### 4.1 整车动力学建模

机器人被简化为一个**轮式倒立摆**模型，包含三个刚体：

1. **驱动轮**（质量 $m_w$，半径 $R$，转动惯量 $I_w$）
2. **摆杆/虚拟腿**（质量 $m_p$，长度 $L$，转动惯量 $I_p$）——由 VMC 等效
3. **机体**（质量 $M$，质心距转轴 $l$，转动惯量 $I_M$）

**状态变量**（6 维）：

$$\mathbf{x} = \begin{bmatrix} \theta \\ \dot{\theta} \\ x \\ \dot{x} \\ \phi \\ \dot{\phi} \end{bmatrix}$$

其中：
- $\theta$：虚拟腿摆杆与竖直方向的夹角
- $x$：驱动轮线位移
- $\phi$：机体 Pitch 角（俯仰角）

**控制输入**（2 维）：

$$\mathbf{u} = \begin{bmatrix} T \\ T_p \end{bmatrix}$$

- $T$：轮毂电机输出力矩
- $T_p$：髋关节输出力矩

通过牛顿-欧拉方程在平衡点线性化，得到状态空间方程：

$$\dot{\mathbf{x}} = A\mathbf{x} + B\mathbf{u}$$

> 具体推导过程见 `simulation/get_k_length.m`，使用 MATLAB 符号计算自动求解雅可比矩阵。

### 4.2 LQR 控制器

#### 4.2.1 原理

LQR（线性二次调节器）通过最小化代价函数：

$$J = \int_0^{\infty} (\mathbf{x}^T Q \mathbf{x} + \mathbf{u}^T R \mathbf{u}) \, dt$$

求解黎卡提方程得到最优反馈增益矩阵 $K$：

$$\mathbf{u} = -K\mathbf{x}$$

$K$ 是 $2 \times 6$ 矩阵，每行对应一个控制输入（轮毂力矩 T 和髋关节力矩 Tp），每列对应一个状态变量。

#### 4.2.2 增益随腿长自适应

由于虚拟腿长 $L_0$ 在运动中会变化（范围 0.065m ~ 0.18m），不同腿长下系统动力学参数不同，需要不同的 $K$ 矩阵。

**离线计算**（MATLAB `get_k.m`）：
1. 在腿长范围 [0.10, 0.21]m 内，以 0.01m 为步长，对每个腿长计算一次 LQR 增益
2. 对 $K$ 矩阵的每个元素（共 12 个）分别用**三次多项式**拟合腿长的函数关系

**在线查表**（STM32 `VMC_calc.c`）：
```c
// 三次多项式拟合：K_ij = a*L^3 + b*L^2 + c*L + d
float LQR_K_calc(float *coe, float len) {
    return coe[0]*len*len*len + coe[1]*len*len + coe[2]*len + coe[3];
}
```

拟合系数存储在 `Poly_Coefficient[12][4]` 数组中。每个控制周期根据当前腿长实时计算 12 个 LQR 增益。

#### 4.2.3 代码中的 LQR 计算

在 `chassisR_task.c` 的 `chassisR_control_loop()` 中：

```c
// 1. 根据当前腿长更新 LQR 增益
for (int i = 0; i < 12; i++) {
    LQR_K[i] = LQR_K_calc(&Poly_Coefficient[i][0], vmcr->L0);
}

// 2. 轮毂电机力矩 T = K1 * x (第一行)
wheel_T = K[0]*(θ - 0) + K[1]*(dθ - 0) + K[2]*(x - x_set)
        + K[3]*(v - v_set) + K[4]*(Pitch - 0) + K[5]*(PitchGyro - 0);

// 3. 髋关节力矩 Tp = K2 * x (第二行)
Tp = K[6]*(θ - 0) + K[7]*(dθ - 0) + K[8]*(x - x_set)
   + K[9]*(v - v_set) + K[10]*(Pitch - 0) + K[11]*(PitchGyro - 0);
```

**注意**：左半边的 LQR 计算中，位移和速度误差方向取反（`x_set - x_filter` 而非 `x_filter - x_set`），Pitch 和 PitchGyro 也取反，这是因为左右腿电机的安装方向相反。

#### 4.2.4 Q/R 矩阵设计

当前使用的参数（来自 `get_k_length.m`）：

$$Q = \text{diag}([1, \; 0.07, \; 10, \; 5, \; 300, \; 0.6])$$

$$R = \begin{bmatrix} 20 & 0 \\ 0 & 1 \end{bmatrix}$$

$Q$ 矩阵对角线含义：
| 索引 | 状态量 | Q 值 | 含义 |
|------|--------|------|------|
| 0 | $\theta$ (腿摆角) | 1 | 腿摆角偏差权重 |
| 1 | $\dot{\theta}$ (腿摆角速度) | 0.07 | 腿摆角速度权重 |
| 2 | $x$ (位移) | 10 | 位移偏差权重 |
| 3 | $\dot{x}$ (速度) | 5 | 速度偏差权重 |
| 4 | $\phi$ (Pitch 俯仰角) | 300 | **最大权重**，保证机体不倒 |
| 5 | $\dot{\phi}$ (Pitch 角速度) | 0.6 | 俯仰角速度权重 |

$R$ 矩阵对角线含义：
| 索引 | 控制量 | R 值 | 含义 |
|------|--------|------|------|
| 0 | $T$ (轮毂力矩) | 20 | 较大值，限制轮毂力矩消耗 |
| 1 | $T_p$ (髋关节力矩) | 1 | 较小值，允许髋关节较大力矩 |

### 4.3 PID 控制器

系统中使用了**多组 PID 控制器**处理 LQR 无法直接处理的自由度：

#### 4.3.1 腿长 PID

**目的**：控制虚拟腿长 $L_0$ 跟踪设定值。

```c
#define LEG_PID_KP  350.0f
#define LEG_PID_KI  0.0f    // 纯 PD 控制
#define LEG_PID_KD  3000.0f
#define LEG_PID_MAX_OUT  90.0f  // 最大输出 90N
```

输出叠加前馈补偿（重力补偿）：
```c
F0 = 13.0f / arm_cos_f32(θ) + PID_Calc(leg, L0, leg_set);
//     ↑ 前馈：mg/cos(θ)      ↑ PD 反馈
```

- 前馈项 `13.0/cos(θ)` ≈ 整车重力沿腿方向的分量（约 1.3kg × 9.8 ≈ 13N）
- PD 反馈项确保腿长收敛到设定值

#### 4.3.2 横滚角 (Roll) PID

**目的**：保持机体左右平衡。通过差异化左右腿的沿腿方向力 $F_0$ 实现。

```c
#define ROLL_PID_KP 100.0f
#define ROLL_PID_KD 5.0f
#define ROLL_PID_MAX_OUT  10.0f
```

计算方式（手动分离 PD，D 项使用角速度而非误差微分）：
```c
roll_f0 = Kp * (roll_set - roll) - Kd * Gyro[1];
```
- 右腿：`F0 = F0 - roll_f0`
- 左腿：`F0 = F0 + roll_f0`

#### 4.3.3 转向 (Yaw/Turn) PID

**目的**：控制偏航角转向。通过给左右轮毂电机施加差速力矩实现。

```c
#define TURN_PID_KP 2.0f
#define TURN_PID_KD 0.2f
#define TURN_PID_MAX_OUT  2.0f
```

```c
turn_T = Kp * (turn_set - total_yaw) - Kd * Gyro[2];
// 右轮: wheel_T = wheel_T - turn_T
// 左轮: wheel_T = wheel_T - turn_T (对称减去)
```

#### 4.3.4 防劈叉 PID

**目的**：防止左右腿摆角差异过大（"劈叉"）。

```c
#define TP_PID_KP 10.0f
#define TP_PID_KD 0.1f
#define TP_PID_MAX_OUT  2.0f
```

```c
theta_err = 0 - (theta_right + theta_left);  // 左右腿摆角之和应为 0
leg_tp = PID_Calc(&Tp_Pid, theta_err, 0.0f);
// 叠加到两腿的 Tp 上
```

#### 4.3.5 PID 实现细节

基础 PID 支持位置式和增量式两种模式（`pid.c`）：

```c
// 位置式 PID
Pout = Kp * error;
Iout += Ki * error;
Dout = Kd * (error - last_error);
out = Pout + Iout + Dout;  // 带限幅
```

高级 PID（`controller.c`）还支持：
- **梯形积分** (Trapezoid Integral)
- **微分先行** (Derivative on Measurement)
- **变速积分** (Changing Integration Rate)
- **输出滤波** (Output LPF)
- **微分滤波** (Derivative LPF)
- **模糊 PID** (Fuzzy PID) —— 根据误差 e 和误差变化率 ec 动态调整 Kp/Ki/Kd

### 4.4 VMC 虚拟腿模型

#### 4.4.1 五连杆正运动学

五连杆结构参数：
- $l_5 = 0.08$m (AE 间距，机体宽度)
- $l_1 = l_4 = 0.075$m (大腿连杆)
- $l_2 = l_3 = 0.14$m (小腿连杆)

已知关节角 $\phi_1, \phi_4$（由电机编码器获取），通过几何关系求解虚拟足端 C 点坐标，进而得到：
- **虚拟腿长** $L_0 = \sqrt{(X_C - l_5/2)^2 + Y_C^2}$
- **虚拟腿摆角** $\phi_0 = \text{atan2}(Y_C, X_C - l_5/2)$
- **LQR 状态量** $\theta = \pi/2 - \text{Pitch} - \phi_0$

#### 4.4.2 雅可比矩阵逆运算

给定虚拟力（沿腿方向力 $F_0$ 和髋关节力矩 $T_p$），通过雅可比矩阵转换为实际关节力矩：

$$\begin{bmatrix} \tau_1 \\ \tau_4 \end{bmatrix} = J^T \begin{bmatrix} F_0 \\ T_p \end{bmatrix}$$

```c
// VMC_calc_2(): 雅可比逆运算
torque_set[0] = j11 * F0 + j12 * Tp;  // 前关节电机力矩
torque_set[1] = j21 * F0 + j22 * Tp;  // 后关节电机力矩
```

#### 4.4.3 离地检测

通过估算虚拟足端支持力判断是否离地：

```c
FN = F0 * cos(θ) + Tp * sin(θ) / L0 + 6.0f;  // 简化模型
if (FN < 5.0f) → 离地
```

离地时的处理策略：
- 轮毂力矩置零，避免轮子空转
- 髋关节只保留腿摆角的 PD 控制
- 位移积分清零，防止落地后的大幅速度修正
- 重新锁定偏航角设定值

### 4.5 卡尔曼滤波状态观测器

#### 4.5.1 目的

轮毂电机编码器只能测量轮子的旋转速度，直接转换的车体速度存在腿部运动的耦合误差。加速度计虽然能测量加速度，但积分存在漂移。卡尔曼滤波融合两者，得到准确的车体速度和位移。

#### 4.5.2 模型

**状态向量**：$\mathbf{x} = [v, \; a]^T$（速度，加速度）

**状态转移矩阵**（dt = 3ms）：

$$F = \begin{bmatrix} 1 & 0.003 \\ 0 & 1 \end{bmatrix}$$

**观测矩阵**：$H = I_{2 \times 2}$

#### 4.5.3 速度解算

观测值的计算（`observe_task.c`）：

```c
// 右腿编码器速度（考虑腿部运动耦合）
wr = -wheel_vel - PitchGyro + d_alpha;           // 轮子相对地面的角速度
vrb = wr * 0.0603 + L0*dθ*cos(θ) + dL0*sin(θ);  // 投影到车体坐标系

// 取左右腿平均
aver_v = (vrb - vlb) / 2.0f;

// 卡尔曼滤波
MeasuredVector[0] = aver_v;         // 编码器解算速度
MeasuredVector[1] = MotionAccel_b[1]; // IMU 体坐标系加速度
```

位移通过速度积分得到：
```c
x_filter = x_filter + v_filter * dt;
```

### 4.6 姿态解算 (Mahony + EKF)

使用 **Mahony 互补滤波器**融合加速度计和陀螺仪数据，解算四元数姿态，得到 Roll/Pitch/Yaw 欧拉角。

关键参数：
- Kp = 1.0f，Ki = 0.0f（纯比例互补）
- 更新周期 1ms

加速度计运动加速度提取：
```c
// 减去重力分量，得到运动加速度
gravity_b = EarthToBody(gravity, q);   
MotionAccel_b = LPF(Accel - gravity_b); // 低通滤波
MotionAccel_n = BodyToEarth(MotionAccel_b, q); // 转换到导航系
```

## 5. 调参指南

### 5.1 LQR 调参 (Q/R 矩阵)

LQR 调参本质是调整 Q 和 R 矩阵，需要在 **MATLAB** 中修改后重新拟合多项式系数，然后**烧录到 STM32**。

#### 调参原则

| Q/R 变化 | 效果 |
|----------|------|
| 增大 Q 的某个对角元素 | 更严格约束对应状态量，响应更快但可能震荡 |
| 减小 Q 的某个对角元素 | 放松对应状态量约束，响应变慢但更平滑 |
| 增大 R 的某个对角元素 | 限制对应控制量输出，节省能量但响应变慢 |
| 减小 R 的某个对角元素 | 允许更大控制力矩，响应更快但电机负担大 |

#### 各状态量调参建议

| 状态量 | Q 增大时的效果 | 典型范围 | 调参优先级 |
|--------|---------------|---------|-----------|
| $\theta$ (腿摆角) | 腿部更"硬"，保持竖直 | 1 ~ 50 | ★★★ |
| $\dot{\theta}$ (腿角速度) | 抑制腿部摆动 | 0.01 ~ 1 | ★★ |
| $x$ (位移) | 位置保持精度更高 | 5 ~ 100 | ★★★ |
| $\dot{x}$ (速度) | 加减速更积极 | 1 ~ 50 | ★★★ |
| $\phi$ (Pitch) | **最关键**：防倒更积极 | 100 ~ 1000 | ★★★★★ |
| $\dot{\phi}$ (Pitch 角速度) | 抑制俯仰震荡 | 0.1 ~ 5 | ★★ |

#### 调参步骤

1. **先调 $Q(5)$**（Pitch 权重）：这是最关键的参数。增大到机器人能稳定站立为止（推荐 200~500 起步）
2. **再调 $Q(3), Q(4)$**（位移和速度权重）：让机器人在受到推力后能回到原位
3. **调 $R(1), R(2)$**：平衡控制性能和电机负载。$R(1)$ 增大→轮毂力矩变小→省电但平衡变差；$R(2)$ 增大→髋关节力矩变小
4. **微调 $Q(1), Q(2), Q(6)$**：减小震荡

#### MATLAB 操作流程

```matlab
% 1. 修改 get_k_length.m 中的 Q/R
Q = diag([1 0.07 10 5 300 0.6]);
R = [20 0; 0 1];

% 2. 运行 get_k.m 获取多项式拟合系数
get_k  % 输出 a11~a26 共 12 组四次多项式系数

% 3. 将系数复制到 chassisR_task.c 的 Poly_Coefficient 数组
```

### 5.2 PID 调参

#### 腿长 PID (LEG_PID)

```c
#define LEG_PID_KP  350.0f  // 腿部刚度
#define LEG_PID_KD  3000.0f // 腿部阻尼
```

| 参数 | 调大效果 | 调小效果 | 建议 |
|------|---------|---------|------|
| KP | 腿更硬，响应快 | 腿更软，响应慢 | 200~500 |
| KD | 阻尼大，不易大幅振荡 | 阻尼小，可能抖动 | 1000~5000 |

> **注意**：这是用于控制沿腿方向虚拟力 $F_0$ 的 PD 控制器，没有积分项（纯 PD）。KD 很大因为腿长变化量很小（单位为 m）。

#### Roll PID

```c
#define ROLL_PID_KP 100.0f

```

- KP 过大：左右晃动频繁；过小：侧倾恢复慢
- KD 使用角速度直接微分，比误差微分更平滑

#### Turn PID

```c
#define TURN_PID_KP 2.0f

```

- KP 过大：转向灵敏但可能摆头；过小：转向迟钝
- MAX_OUT = 2.0 限制差速力矩，防止急转弯导致失衡

#### 防劈叉 PID

```c
#define TP_PID_KP 10.0f
#define TP_PID_KD 0.1f
```

- 保持左右腿摆角一致的补偿控制器

### 5.3 卡尔曼滤波调参

```c
float vaEstimateKF_Q[4] = {0.1f, 0.0f, 0.0f, 0.1f};    // 过程噪声
float vaEstimateKF_R[4] = {100.0f, 0.0f, 0.0f, 100.0f}; // 测量噪声
```

| 参数 | 调大效果 | 调小效果 |
|------|---------|---------|
| Q (过程噪声) | 更信任观测值，滤波器响应快但噪声大 | 更信任模型预测，滤波器响应慢但平滑 |
| R (测量噪声) | 更信任模型预测，滤波慢延迟大 | 更信任观测值，可能跟随噪声 |

- **Q/R 比值**决定滤波器的"软硬"程度
- 当前 Q=0.1, R=100，即 Q/R=0.001，说明更信任速度模型预测，适合编码器测量噪声较大的场景
- 如果速度跟踪滞后，适当增大 Q 或减小 R

### 5.4 调参实操流程

推荐的调参顺序：

```
1. 基础稳定
   ├── 先用 MATLAB 计算一组保守的 LQR K 矩阵
   ├── Pitch 权重 Q(5) 给大 (500+)
   ├── R 矩阵给大 (R(1)=90, R(2)=25) → 限制力矩输出
   └── 确保机器人能基本站立

2. 腿部调参
   ├── 调 LEG_PID_KP/KD → 让腿长稳定在设定值
   ├── 调防劈叉 PID → 两腿摆角保持一致
   └── 调 Roll PID → 消除侧倾

3. 运动性能
   ├── 减小 R(1) → 轮毂电机可以输出更大力矩
   ├── 增大 Q(3), Q(4) → 更积极地跟踪速度/位置
   ├── 调 Turn PID → 转向灵敏度
   └── 调卡尔曼 Q/R → 速度估计的精度和延迟

4. 精细优化
   ├── 调 Q(1), Q(2) → 腿部摆动抑制
   ├── 调 Q(6) → 俯仰角速度阻尼
   └── 重新拟合多项式系数烧录
```

## 6. 外界干扰应对策略

### 6.1 LQR 的抗干扰能力

LQR 本身是一个**状态反馈控制器**，其抗干扰能力来自：

1. **全状态反馈**：6 个状态量同时参与控制计算，任何状态偏离都会立即产生修正力矩
2. **最优性**：在给定 Q/R 下，LQR 提供的是理论最优的线性反馈增益
3. **相位裕度保证**：LQR 保证至少 60° 的相位裕度和无穷大增益裕度（针对线性系统）

**局限性**：
- LQR 基于线性化模型，当 Pitch 角偏离平衡点过大（>30°）时，线性化假设不成立，控制力不足
- 没有积分环节，存在稳态误差（虽然由速度闭环的 PID 部分补偿）

### 6.2 各种干扰场景及应对

#### 推力干扰（被推一下）
```
干扰 → Pitch 突变 → LQR K[4]*Δφ + K[5]*Δφ̇ 产生大力矩
     → 速度突变 → LQR K[3]*Δv 产生修正力矩
     → 位移积分 → LQR K[2]*Δx 缓慢回到原位
```
- Q(5) 越大，Pitch 恢复越快
- Q(3) 越大，位置回正越积极

#### 坡道/倾斜地面
```
持续倾斜 → Pitch 持续偏移 → LQR 持续输出力矩对抗
         → 速度/位移积分补偿稳态误差
```
- 如果需要更好的坡道性能，可考虑在速度环添加积分项

#### 离地/腾空
```
离地检测 → FN < 5N → 判定离地
       → 轮毂力矩置零 (防空转)
       → 髋关节只保留腿角 PD
       → 位移/偏航角重置
落地后 → 恢复正常控制
```

#### 侧向干扰（被横推）
```
横推 → Roll 角变化 → Roll PID 产生左右腿 F0 差异
    → 一腿伸长一腿缩短 → 机体回正
```

#### 单侧台阶/不平地面
```
一腿着地一腿悬空 → 腿长 PID 自动调节
                → 离地检测保护
                → 防劈叉 PID 防止两腿角度差异过大
```

### 6.3 提高抗干扰性能的方法

1. **增大 LQR 的 Q 矩阵对角元素**：对关键状态量（Pitch、速度）增大权重
2. **添加积分项**：在 LQR 外层叠加位移/速度的积分控制器消除稳态误差
3. **调整卡尔曼滤波 Q/R**：提高状态观测器的响应速度（增大 Q 或减小 R）
4. **增益调度**：当前已实现的腿长自适应 LQR 就是一种增益调度方法；可进一步根据速度等参数调度增益
5. **优化离地检测算法**：使用更完善的支持力估算公式（注释中有被注释掉的完整版本，包含了惯性力项）
